To push to Github
-----------------

`git add .`
`git commit -m 'message'`
`git push origin master`


To push to Heroku
-----------------

(add stuff first)
`git push heroku master`



To run on your local machine
----------------------------

cd into folder
`shotgun config.ru`
Go to http://localhost:9393 in your browser
Type `ctrl - c` to shut down server
